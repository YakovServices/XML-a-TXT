Guía de uso del Transformador XML a TXT

Por la estructura del programa es importante que al poner la ruta de los archivos .xml en la misma carpeta que el programa, por que si no no va a detetcar los .xml.

Este programa te permite transformar archivos XML a archivos de texto y gestionar archivos XML.

Paso 1: Ejecutar el programa

Abre la terminal y navega a la carpeta donde se encuentra el archivo Iniciar.py.

Paso 2: Ejecutar el script Iniciar.py

Ejecuta el siguiente comando:

python3 Iniciar.py

Paso 3: Usar el comando "ayuda"

Para ver la lista de comandos disponibles, ingresa ayuda y presiona Enter.

Paso 4: Usar el comando "inspeccionar"

Para detectar archivos .xml en una ruta específica y añadirlos al archivo nobres-de-los-xml.txt, ingresa inspeccionar y presiona Enter. Luego, ingresa la ruta cuando se te solicite.

Paso 5: Usar el comando "iniciar"

Para ejecutar el script xml-a-txt-completo.py, ingresa iniciar y presiona Enter. Luego, ingresa el nombre del archivo de texto de salida cuando se te solicite.

Paso 6: Salir del programa

Para salir del programa, ingresa salir y presiona Enter.
